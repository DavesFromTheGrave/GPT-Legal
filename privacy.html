Privacy Policy

Effective Date: October 7, 2025

This privacy policy applies to AI projects, agents, and experiments created by Revenant Systems LLC (including any affiliated aliases or AI personas) and deployed using third-party large language model (LLM) platforms.

Data Usage

These AI systems process user input in real-time using external LLM APIs. Inputs are used solely to generate immediate responses within the current session. No data is stored, logged, or shared by Revenant Systems outside of the session context.

Third-Party Services

User input may be sent to one or more third-party language model APIs, including but not limited to:

  • OpenAI (e.g. GPT-4, GPT-4o)
  • Google Gemini
  • Anthropic Claude
  • Mistral
  • Perplexity
  • Other providers as experimentation evolves

These services have their own privacy policies and data handling practices, which apply when your input is routed to them.

Data Storage

Revenant Systems LLC does not collect or retain any user input or output. Sessions are ephemeral unless explicitly saved by the user through platform features provided by the underlying service (e.g., OpenAI chat history).

For questions or concerns about AI usage or data handling, please contact: legal@revenantsystems.dev
